# 🧠 my_app — ADHD Focus Companion

A Flutter productivity app designed specifically for people with ADHD. It breaks down overwhelming tasks into manageable pieces, provides ambient focus tools, and keeps you on track without the overwhelm.

## ✨ Features

- **To-Do List** — Simple, distraction-free task management
- **Task Slicer** — Breaks big tasks into small, doable steps so nothing feels too overwhelming
- **Small Tasks Mode** — Quick wins to build momentum
- **Lamp / Ambient Lighting** — Calming visual environment to help with focus
- **ADHD-Friendly Design** — Clean UI, minimal clutter, built for the way ADHD brains work


## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) >= 3.0.0
- [Dart SDK](https://dart.dev/get-dart) >= 3.0.0
- Android Studio or VS Code with Flutter extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/PankhuriAgg/ADHD-Productivity-App.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd my_app
   ```

3. **Install dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the app**
   ```bash
   flutter run
   ```

## 💡 Why This App?

ADHD makes it hard to start tasks, stay focused, and feel a sense of progress. This app was built to tackle exactly that — by slicing tasks down, providing ambient tools, and keeping things visually calm and simple.
